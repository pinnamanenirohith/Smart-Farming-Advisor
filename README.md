# Smart Farming Advisor

## Overview

Smart Farming Advisor is an **AI-powered chatbot project** developed using **IBM Watson Assistant** to support farmers with essential agricultural information.

The chatbot is designed to provide guidance related to:

* Crop recommendations
* Market (mandi) prices
* Pest control suggestions
* Weather forecasts

The assistant supports multilingual interactions, including **English and Telugu**, making agricultural guidance more accessible to local farmers.

## Problem Statement

Farmers often face challenges in accessing timely and reliable information regarding crop planning, pest control, market pricing, and weather conditions.

This project aims to provide a conversational AI-based solution that simplifies access to farming-related information through an intelligent virtual assistant.
